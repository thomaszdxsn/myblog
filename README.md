# myblog

## api -- version1.0

资源

uri: `/api/v1/<resource>/(<id>)*`

- user
- category
- post
- tag
- image

### patch

- 资源新建/更新成功后会返回201状态码(2018-1-15)
- 资源删除成功后会返回204状态码(2018-1-15)
- 为资源的访问/操作加入验证登录的要求，如果没有登录返回401状态码(2018-1-15)
